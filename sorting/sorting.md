## Stable

Original list:

```
[("Alice", 85), ("Bob", 90), ("Charlie", 85), ("David", 80)]
```

sorting with stable sort:

```
[("Bob", 90), ("Alice", 85), ("Charlie", 85), ("David", 80)]
```

Alice and Charlie both have 85, but Alice remains before Charlie because that was their original order.

Stable sorts

* Merge
* Bubble
* Insertion
